# Divine (Research & Exploration)

Channel the spirits to research, investigate, or explore a topic. Used for gathering information before implementation.

## Topic: $ARGUMENTS

Use the Explore agent (Task tool with subagent_type="Explore") to investigate the topic thoroughly.

### Divine Modes

- **Codebase divination**: Search for patterns, understand architecture
- **Documentation lookup**: Use Context7 for library docs
- **Web research**: Search and fetch external information
- **Requirements analysis**: Understand what needs to be built

### Output Format

Provide findings in a clear, structured format:
1. Key discoveries
2. Relevant files/locations
3. Recommendations for next steps
4. Any concerns or risks identified

Begin the divination ritual for the specified topic.
